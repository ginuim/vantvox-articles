# 孔子：失败者与塑造者

## 一个"丧家犬"的文明回响

公元前489年，一个六十二岁的老人带着他的学生们在陈国和蔡国之间的野外艰难前行。他们已经被困七天，没有食物可以充饥。有些人病倒了，有些人开始低声抱怨。这位老人却依然端坐在地上，弹琴唱歌，仿佛这一切都与他无关。

这是《论语》中记载的一个场景。司马迁在《史记》中引用了孔子学生子贡的话，说当时有人"诟病"（嘲笑、辱骂）孔子，而孔子始终坚持自己的学说。司马迁用了一个精辟的比喻来形容孔子此时的状态：**"然孔子欲以明道德仁义之途，栖栖然而与世作罗笼，终以不容于世。"** 意思是，孔子想要阐明道德仁义之道，忙忙碌碌地与世俗周旋，却始终不被世道所容。

这个场景发生的二十四年后（前479年），孔子在鲁国首都曲阜的家中孤独地死去。他的身份是：一个没有正式官职的退休教师，一个四处游说却始终得不到重用的政客，一个被多个诸侯国礼貌拒绝的老人。

然而，就是这个"失败者"的遗体，在两千五百多年后，被安放在总面积约3.5万平方米的孔庙中，接受着每年超过数百万人的瞻仰。他的思想成为东亚文明的核心支柱，影响了地球上超过十五亿人的生活。他的名字，成为人类历史上被提及最多的名字之一。

这是一个关于**逆转**的故事：一个生前彻底失败的人，如何在身后两千五百多年里逐步塑造了一个伟大文明？

## 一、时空坐标：礼崩乐坏的春秋末期

### 时间坐标：动荡与秩序的千年变局

孔子的时代，是中国历史上一个独特的时间节点。

让我们把时间放在更大的尺度上审视。孔子出生前约五百年，周武王在牧野之战中击败商朝，建立周朝，实行分封制和宗法礼乐制度。孔子出生前约三百年，周王室开始衰落，诸侯国之间攻伐不断，礼乐制度逐渐瓦解。孔子所生活的春秋末期（约前770年—前476年），正是这个"礼崩乐坏"过程加速的阶段。

具体到孔子的生命历程：

- **前551年**，孔子出生于鲁国昌平乡（今山东曲阜），父亲叔梁纥是鲁国贵族，但已家道中落。
- **前548年**（3岁），父亲去世，与母亲颜徵在相依为命。
- **前537年**（14岁），开始学习礼乐，对古典文献产生浓厚兴趣。
- **前532年**（19岁），娶宋国女子亓官氏为妻。
- **前525年**（26岁），在鲁国创办私人学校，打破了"学在官府"的传统。
- **前517年**（34岁），因鲁国内乱，赴齐国，第一次尝试从政。
- **前514年**（37岁），返回鲁国，继续教学和整理古文献。
- **前500年**（51岁），任鲁国中都宰（小的地方长官），治理有方。
- **前498年**（53岁），任鲁国司寇（司法长官），参与削弱贵族势力的"堕三都"行动。
- **前497年**（54岁），因与鲁国权贵季桓子的矛盾，开始长达十四年的周游列国之旅。
- **前484年**（67岁），返回鲁国，整理古文献，教授学生。
- **前479年**（73岁），孔子去世。

这七十三年的人生，孔子并没有取得任何世俗意义上的"成功"。他做过最大的官是司寇，相当于今天的司法部长，但任职时间不超过三年。他周游列国十四年，游说了卫、曹、宋、郑、陈、蔡、楚等多个诸侯国，却没有得到任何一个国君的重用。

### 空间坐标：黄河流域的文明版图

孔子的活动范围，主要集中在黄河中下游地区，以今天的河南省、山东省为中心。

在孔子生活的时代，中国还远未形成一个统一的国家。周王室名义上是天下共主，实际上只控制着洛阳周围的一小片领地。诸侯国遍布各地，其中较大的有齐、晋、楚、吴、越、秦、宋、鲁、郑、陈、蔡等。这些诸侯国之间相互攻伐，争夺霸权，构成了一个复杂的国际体系。

孔子出生于鲁国，这是周公旦的封地，保存了较为完整的周礼传统。鲁国虽小，却在文化上具有特殊地位——它是周礼在东方的保存者。孔子的思想根基，正是这种深厚的礼乐传统。

但也正是在鲁国，孔子遭遇了他政治生涯中最惨重的失败。前498年，他与鲁国实际执政者季康子合作，试图拆除三个贵族家族（季孙氏、孟孙氏、叔孙氏）的城池，以加强公室权力。结果是"堕三都"行动半途而废，孔子与贵族集团彻底决裂，被迫离开鲁国。

### 问题坐标：一个无法回答的时代之问

孔子面对的核心问题，是那个时代所有思想家都在思考的问题：**如何重建社会秩序？**

在周礼崩溃、诸侯争霸的乱世中，不同的思想家给出了不同的答案：

- **法家**（如商鞅、韩非）主张以法治国，强调严刑峻法和君主集权。
- **道家**（如老子、庄子）主张无为而治，放弃人为努力，回归自然。
- **墨家**（墨翟）主张兼爱、非攻、尚贤，反对战争和等级制度。
- **儒家**（孔子）则主张恢复周礼，通过道德教育和礼仪规范来重建社会秩序。

孔子的方案的核心，是"仁"与"礼"的统一。"仁"是内在的道德情感，"礼"是外在的行为规范。他认为，只要每个人都修养自己的品德，遵守应有的礼仪，社会就能恢复和谐。

这个方案在理论上很美好，但在实践中却处处碰壁。因为春秋末期的诸侯们需要的不是道德说教，而是切实的军事实力和政治谋略。孔子的学说，对于急于求成的诸侯们来说，太"慢"了，也太"软"了。

## 二、生前失败： 一个"被礼貌拒绝"的思想家

### 从政生涯：三次重大挫败

孔子的政治生涯，可以概括为三次重大挫败。

**第一次挫败：齐国之行（前517年）**

孔子34岁时，因鲁国内乱（鲁昭公被季孙氏驱逐）而避难齐国。在齐国，他得到了齐景公的接见，并向齐景公提出了"君君、臣臣、父父、子子"的正名主张——意思是国君要像国君，臣子要像臣子，父亲要像父亲，儿子要像儿子。齐景公听了很高兴，说"善哉！信如君不君，臣不臣，父不父，子不子，虽有粟，吾得而食诸？"——说得对啊！如果君不像君，臣不像臣，父不像父，子不像子，即使有粮食，我能吃得着吗？

齐景公似乎被孔子说动了，打算给他封地重用他。但据《史记》记载，晏婴（齐国名相）坚决反对，认为孔子的那一套"礼"过于繁琐，不切实际。齐景公最终打消了念头，"敬见孔子，不问其礼"，态度变得冷淡。孔子只好离开齐国。

这是孔子第一次被"礼貌拒绝"。齐景公没有对他怎么样，但也没有重用他。

**第二次挫败：鲁国司寇生涯（前500年—前497年）**

这是孔子一生中唯一的正式高官经历。他任鲁国司寇期间，曾迫使齐国归还了侵占鲁国的郓、龟阴等地（《史记》称"夹谷之会"）。这可能也是他一生中最"风光"的时刻。

但好景不长。随着他参与"堕三都"行动，触犯了贵族利益，季桓子开始疏远他。据《论语》记载，有一次季桓子接受了齐国送来的女乐（歌伎），连续三天不处理政事。孔子对此极度失望，离开了鲁国。

这一走，就是十四年。

**第三次挫败：周游列国（前497年—前484年）**

这是孔子人生中最艰难的时期。他带着弟子们辗转于卫、曹、宋、郑、陈、蔡、楚等国，推行自己的政治主张，但始终没有得到任何国君的重用。

在陈国和蔡国之间被困七天的经历，是这段历程的缩影。据《史记》记载，孔子一行被困，"绝粮，从者病，莫能兴"。子路对此表示怀疑："君子亦有穷乎？"——君子也会走投无路吗？孔子回答："君子固穷，小人穷斯滥矣。"——君子即使走投无路也坚持原则，小人一旦走投无路就无所不为了。

在卫国，孔子被怀疑有野心，遭到监视。在宋国，他差点被宋国司马（军事长官）杀掉。在郑国，他与学生失散，被形容为"累累若丧家之狗"——一只无家可归的狗。

这就是孔子"失败"的具体图景：不是被迫害致死（苏格拉底的命运），而是被礼貌地拒绝，一次又一次。

### 比较视角：苏格拉底的"失败"与"成功"

把孔子的命运与同时代的古希腊哲学家相比，是一件很有趣的事情。

苏格拉底（前470年—前399年），比孔子小约80岁。两个人的命运有相似之处：

| 比较维度 | 孔子 | 苏格拉底 |
|---------|------|----------|
| 时代 | 春秋末期（约前551年—前479年） | 雅典民主时期（约前470年—前399年） |
| 身份 | 教师、思想家 | 教师、哲学家 |
| 主要活动 | 游说诸侯推行政治主张 | 在雅典广场与人辩论 |
| 生前遭遇 | 被多个诸侯国礼貌拒绝 | 被雅典民主法庭判处死刑 |
| 死亡方式 | 病逝（73岁） | 服毒自尽（71岁） |
| 身后影响 | 成为中国思想文化核心 | 成为西方哲学奠基人 |

苏格拉底的"失败"更为极端——他被自己城邦的公民投票判处死刑。但正是这种"悲剧性死亡"，使他成为西方思想史上的殉道者。柏拉图继承他的思想，创立 Academy（柏拉图学园），西方哲学由此发端。

孔子的"失败"则更为温和——他只是被"礼貌拒绝"而已。没有一个国君要杀他，没有一场审判定他的罪。他只是"不被需要"。

这或许解释了为什么孔子在身后花了两千多年才登上"至圣先师"的巅峰，而苏格拉底在生前就已经是雅典的精神象征。

哈佛大学中国历史学教授 Michael Puett（浦慕明）在其著作《The Path: A New Way to Think About Everything》中评论道："苏格拉底和孔子都试图改变他们的社会，但方式截然不同。苏格拉底通过质疑和辩论，挑战雅典人的常识；孔子通过教育和礼仪，试图重建社会规范。两个人的'失败'都深刻影响了后世，但影响的方式很不一样。"

## 三、身后成功： 从"丧家犬"到"至圣先师"

### 身后两百年的默默无闻

孔子死后，他的思想并没有立即产生巨大影响。

他的学生们散布各地，继续传播他的学说，但影响力有限。在战国时期（前475年—前221年），儒学只是诸子百家中的一家，与道家、墨家、法家、名家等并存。真正使儒学发扬光大的，是战国末期的**荀子**和西汉的**董仲舒**。

荀子（前313年—前238年）是儒家重要的思想家，他吸收了法家的某些观点，对儒学进行了改造。但真正让儒学成为官方意识形态的，是董仲舒。

### 罢黜百家：儒学的登顶之路

公元前134年，西汉皇帝刘彻（汉武帝）采纳了董仲舒的建议，"罢黜百家，独尊儒术"。从此，儒学从一家之言变成了国家意识形态，孔子也被逐步神化。

以下是孔子地位变迁的时间线：

- **前195年**：汉高帝刘邦过鲁国，"以太牢祀孔子"，这是帝王祭祀孔子的开始。
- **前136年**：汉武帝设五经博士，儒学成为官方学问。
- **前51年**：汉宣帝召开石渠阁会议，评定儒学经典。
- **元年**：汉平帝追封孔子为"褒成宣尼公"，这是孔子第一个官方封号。
- **492年**：北魏孝文帝封孔子为"文圣尼父"。
- **605年**：隋炀帝封孔子为"先师尼父"。
- **玄宗开元年间（8世纪）**：唐玄宗封孔子为"文宣王"。
- **1104年**：宋徽宗封孔子为"至圣文宣王"。
- **1415年**：明成祖封孔子为"至圣先师"。
- **1912年**：中华民国临时政府撤消孔子的祭孔地位。
- **1956年**：中华人民共和国国务院将孔子诞辰（9月28日）列为教师节。
- **2014年**：中国国家主席习近平表示要"大力弘扬优秀传统文化"。

这个时间线揭示了一个重要事实：孔子的"成功"不是一蹴而就的，而是经过了长达两千多年的累积。每一次朝代更替、每一次文化转型，孔子都会被重新诠释、重新定位。

### 跨文化影响：东亚文明的共同资产

孔子的思想不仅在中国本土产生了深远影响，还传播到东亚其他地区，成为整个东亚文明的核心价值体系。

**朝鲜半岛**：儒学在朝鲜半岛的影响可以追溯到三国时代（新罗、百济、高句丽）。朝鲜王朝（1392年—1897年）以儒学为官方意识形态，建立了系统的科举制度和书院制度。孔子思想深刻影响了韩国的社会结构和文化心理。

**日本**：儒学在公元6世纪传入日本，对日本的政治制度、教育体系和社会伦理产生了深远影响。明治维新时期，虽然西方思想大量涌入，但儒学中的某些核心价值（如忠、义、礼）仍然发挥着重要作用。

**越南**：儒学通过科举制度深入越南社会，塑造了越南的传统价值观。

哈佛大学燕京学社社长 Eugene Park（朴在渊）在接受采访时曾说："儒学不只是一个中国的现象，它是整个东亚的共同文化遗产。在韩国、日本、越南，孔子都被视为重要的文化导师。"

## 四、争议与局限：一个立体的人的AB面

### 思想体系的争议

孔子的思想体系在后世产生了广泛争议，主要集中在以下几个方面：

**1. 等级制度的正当性**

儒学强调"君君、臣臣、父父、子子"，这被批评为维护封建等级制度。批判者认为，这种思想强化了权威，压抑了个人的独立性和创造力。

美国汉学家 James Legge（理雅各，19世纪著名汉学家）在翻译《论语》时评论道："孔子的学说在实践中往往被用作维护专制的工具，但这不是孔子本人的初衷。"

**2. 女性地位的忽视**

《论语》中关于女性的言论甚少，且有"唯女子与小人为难养也"之句，历来被视为孔子歧视女性的证据。尽管有学者为孔子辩护，认为此句的"女子"可能特指某些人，但儒学传统上对女性地位的忽视是不争的事实。

**3. 保守与创新的张力**

孔子自称"述而不作"（传承而不创新），强调对周礼的继承。这被批评为思想保守，阻碍了社会变革。但也有学者认为，孔子的"述"中自有"作"，他对周礼的诠释本身就包含着创新。

### 历史评价的两极化

孔子的历史评价呈现两极化的特征：

- **正面评价**：汉代以后，孔子被尊为"至圣先师"、"万世师表"，成为中国文化的象征。
- **负面评价**：五四运动时期，孔子被视为"吃人的礼教"的代表，遭到激烈批判。文化大革命时期，孔子被作为"复古主义"的代表受到冲击。

英国汉学家 Arthur Waley（韦利）在其译本《论语》的导言中写道："孔子不是一个创造性的天才，但他是一个卓越的教师和阐释者。他把一套复杂的生活艺术用自己的语言表达出来，这套语言简洁、优雅、充满智慧。"

## 结语：让失败与成功对话

公元前479年，孔子在孤独中死去。他的学生们把他的言论整理成《论语》，这部书只有约一万五千字，却成为人类历史上被阅读最多的书籍之一。

他生前没有担任过真正重要的官职，没有实现自己的政治理想，没有看到一个稳定的中国的出现。他的学说在当时被诸侯们视为"不切实际"，他本人被形容为"丧家之狗"。

但他身后，两千五百多年的时间里，他的思想成为：
- 中国历代王朝的官方意识形态
- 东亚文明的核心价值体系
- 世界上超过十五亿人的精神资源
- 联合国教科文组织认定的世界文化遗产

这是一个关于**时间**的故事。孔子的成功，不是他生前预见的，也不是他生前追求的。他的思想在时间的河流中被不断诠释、不断改造、不断赋予新的意义。每一次朝代更替、每一次文化危机、每一次社会转型，人们都会回到孔子，从他的言论中寻找答案。

苏格拉底死于公元前399年，被雅典公民投票处死。孔子死于公元前479年，病逝于家中。两个同时代的思想家，以不同的方式"失败"了，又以不同的方式"成功"了。他们的"失败"是真实的，他们的"成功"也是真实的。

也许，这正是历史的复杂之处：**一个人无法选择自己的命运，但他的思想可以选择自己的未来。**

孔子成功了？不，是时间成功了。
